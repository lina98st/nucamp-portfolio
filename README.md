# Alina Portfolio

This is my personal portfolio project created for the Nucamp Coding Bootcamp.  
The site is built using **HTML**, **CSS**, and **Bootstrap 5**.

## Overview

The portfolio includes:

- A clean and responsive layout
- Sections for About, Projects, and Contact
- Custom pastel purple color palette
- A personal image (`alina.jpg`) in the About section
- Example project cards (placeholders for future work)

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5 (via CDN)
